Interoch Numbers Theory: A Comprehensive Framework

The Interoch Numbers Theory explores sets of integers closed under specific digit-wise operations. If S is such a set and ∘ is the operation, then for every n in S, applying ∘ to its digits yields another number in S.

1. Foundational Definitions

Let d(n) denote the digits of n.

An Interoch set satisfies:

∀n ∈ S, ∘(d(n))∈S

1.1 Closure(∘):

The operation ∘ ensures no escape from the set S.

2. Core Interoch Sets

  • Addinteroch – Sum of digits.

    • Example: 58 → 5 + 8 = 13.

  • Subinteroch – Absolute difference of digits.

    • Example: 94 → |9 − 4| = 5.

  • Neutinteroch – Mix of addition and subtraction.

    • Example: 47 → (4 + 7) − |4 − 7| = 11 − 3 = 8.

  • Mulinteroch – Product of digits (Order Sensitive).

    • Example: 81 → 8 / 1 = 8

  • Divinteroch – Closed under digit division.

    • Example: 47 → (4 + 7) − |4 − 7| = 11 − 3 = 8.

3. Axiom, Theorems and Properties

3.1 Axioms:

Axiom 1 (Closure Under Digit Operations)
For any Interoch Set S and operation ∘,
∀n ∈ S, ∘(d(n)) ∈ S,
where d(n) extracts the digits of n, and ∘ is a well-defined digit-wise operation (e.g., sum, product, shift).

Corollary:

  • If ∘ is deterministic (e.g., digit sum), S is uniquely closed.
  • If ∘ is non-deterministic (e.g., signed digit difference), S must contain all possible outputs.

Axiom 2 (Minimality Condition)
An Interoch Set S is minimal if:
1. S is closed under ∘.
2. No proper subset of S satisfies (1).

Examples:
{6, 60} is minimal for Addinteroch.
{0} is trivial-minimal for Subinteroch.

Axiom 3 (Operation - Independent Invariance)
An Interoch Set S may satisfy multiple operations simultaneously (e.g., additive and multiplicative closure).
Example: {1} is closed under digit sum (1 → 1) and digit product (1 → 1).

Structural Principles

Principle of Non-Triviality:
A non-trivial Interoch Set must contain at least two numbers linked dynamically (e.g., 38 → 6).

Principle of Maximality:
A maximal Interoch Set cannot be extended without violating closure.

3.2 Minimal and Maximal Sets:

Minimal:- Smallest non-trivial closed set.

Example: {6,60} is minimal for Addinteroch.

Maximal:- Largest possible closed set (ofter infinite).

Example: All numbers divisible by 9 for Addinteroch.

3.3 Theorem - 1:

Any Addinteroch set must share a digital root (mod 9).

Proof: Digit sums preserve modulo 9.

3.5 Fixed Points and Invarience:

Single - digit numbers are fixed points for most operations.

Non - Trivial invarience: {145} for factorial digit sum (1! + 4! + 5! = 145)

4. Advanced Constructs

4.1 Positional Interochs:

  • Swappinteroch – Closed under digit swaps.

    • Example: {13,31}.

  • Splitinteroch – Operate on grouped digits.

    • Example: 123 → 1 + 23 = 24.

4.2 Recursive Interochs:

  • Fractalinteroch – Recursive digit sums.

    • Example: 987 → 24 → 6.

  • Loopinteroch – Numbers cycle back to themselves.

    • Example: 08 → 0 + 8 = 8.

4.3 Compositional Interochs:

  • Multiinteroch – Chains of operations.

    • Example: 14 → 1 + 4 = 5; 5 × 2 = 10.

  • Groupinteroch – Segment and then operate rule.

    • Example: 1242 → 12 + 42 = 54.

4.4 Transformational Interochs:

  • Codeinteroch – Digt shifts.

    • Example: +2 to each digit in 123 → 345.
